Testa att klicka runt bland bilderna

Nu ska vi göra en "bildkarusell" där vi med hjälp av JavaScript kan klicka oss runt bland bilder.
Det första vi ska göra är att skapa "lådan där bilderna ska vara" 

          
            <div class="bildKarusell">
              <button class="btn btnLeft" onclick="plusDivs(-1)"> <i class="fas fa-angle-left"></i></button>
              <button class="btn btnRight" onclick="plusDivs(1)"> <i class="fas fa-angle-right"></i></button>  
              <img src="länk till bild1" class="rounded">
              <img src="länk till bild2" class="rounded">
              <img src="länk till bild3" class="rounded">
            </div>

klassen "rounded" är bara för att bilderna ska se ut på ett visst sätt, här får ni göra lite som ni vill.
Om ni kollar på knapparna i början så ser ni att de har ett attribut onclick="plusDivs()" vi kommer prata mer om denna funktion sen. 

          
          /* BildKarusellen har display flex för att bilderna ska rätas till i behållaren */
            .bildKarusell{
              margin-top: 30px;
              width: 400px;
              display: flex;
              align-items: center;
              position: relative;
            }
            
          /* Bilderna i karusellen måste sättas till samma storlek som behållaren de är i */
          /* När vi skriver "img" i slutet betyder det att alla bilder i .bildkarusell ska påverkas */
            .bildKarusell img{
              width: 400px;
              height: 400px;
            }

Ta tid på er att verkligen förstå vad som händer i den här funktionen!
Det kan vara väldigt svårt att läsa kod som någon annan har skrivit så fråga om hjälp om det är något som du inte förstår. 

          
            function displayImg(n){
              var i;
              var images = document.getElementsByClassName("rounded"); /* Här hämtar vi alla element som har klassen rounded */
              if (n < images.length){slideIndex = 1}  /* Om nr på bilden vi vill visa överstiger antalet bilder vi har vill vi sätta slideIndex=1  */
              if (n < 1){slideIndex = images.length} /* Om nr på bilden vi vill visa är för lågt så ska vi kolla på den sista bilden */

              for (i=0; i < images.length; i++){  /* Här går vi igenom hela listan med bilder */
                images[i].style.display = "none" /* här sätter vi display = "none", vilket gör att  bilden inte visas */ 
              }
              /* I slutet av av funktionen saknas det lite kod för att faktiskt visa rätt bild */
              /* Använd bild.style.display = "block", för att visa en bild */
              /* Tips! Det är variabeln slideIndex som ska bestämma vilken bild som visas */
              /* Försök tänk ut hur du väljer så att rätt bild visas, fråga om tips av lärare eller kompisar om det behövs! */
            }

Vi får inte glömma funktionen plusDivs() som kallas på när vi trycker på en knapp! 

          
            function plusDivs(n){  /* Som ni ser är det enda den här funktionen gör är att den lägger till ett värde på slideIndex */
              displayImg(slideIndex += n);  /* Det man behöver göra är alltså att skicka in ett värde "n" när man kallar på funktionen */
            }

Det sista vi måste göra är att skapa variabeln slideIndex och sedan kalla på funktionen som drar igång allting 

          
            var slideIndex = 1;
              displayImg(slideIndex);

Nu ska vi prova att kombinera lite JavaScript med Css-animationer!
om ni kollar på HTML-koden så kan ni se att vi lägger en framsida och en baksida i samma div med klassen "flipCard"
Lägg märke till att vissa divs innehåller flera klasser t.ex har en div klasserna "cardFront" och "flipBack"
En annan div har klasserna "cardBack" och "flipped" 
          
            <div class="flipCard" onclick="flip(this)">
              <div class="cardFront flippBack"> /* Kortetes framsida */
                <img class="cardImg" src="https://cdn.glitch.com/a6d428e3-0f8b-485e-bc0f-2da614792684%2Ftacos.jpg?1528749954770">
              </div>
              
              <div class="cardBack flipped" onclick="flip()"> /* Kortets baksida */
                <p>Recept Tacos</p>
              </div>
            </div>

Klassen flipped används för att vända på ett kort
Ni behöver skriva klassen flippedBack också, den ser nästan exakt likadan ut men då ska rotatione vara till 0 grader. 

          
            .flipped{
              transform: rotateY(180deg); /* Här gör vi en rotation i 180 gradet, så den hamnar alltså så att baksidan "syns" */
              transition: transform 0.5s ease-in-out;  
            }

flipCard-klassen har bara några enkla egenskaper 

          
          .flipCard{
            width: 150px;
            height: 150px;
            margin: auto;
          }

bilderna i flipKorten vill vi ha lika stora som behållaren den är i 

          
          .cardImg{
            width: 150px;
            height: 150px;
            border-radius: 10px;
            border: solid 2px black;
            }

Några egenskaper som kanske är nya i cardFront är perspective och backface-visability
Perspective används för att kunna göra 3d-animationen senare
backface-visability används för att man inte ska kunna se baksidan på korten när de vänds

Ge samma egenskaper till cardBack klassen efter det här! 

          
          /* Alla klasser cardFront i classen flipCard */
          .flipCard >.cardFront{
            width: 150px;
            height: 150px;
            position: absolute;
            perspective: 600px;
            backface-visibility: hidden;
            }

Nu ska vi börja kolla på Javascripten för den här uppgiften
Om ni kollar tillbaka på HTML-koden tidigare och kollar vad som händer när man klickar på "flipCard"-diven så ser vi onclick = "flip(this)"
flip( ) är funktionen vi ska skapa. När vi skickar in argumentet "this" i en funktion så betyder det att vi skickar in själva elementet vi är på.
Vi skickar alltså in allting som flipCard-diven har i sig. Det är viktigt att komma ihåg när vi kollar på funktionen. 

          
            function flip(element){ /* När vi skriver element.children så får vi tillgång till den inskickade elementets alla barn */
              /* Ett barn till en div är allting som ligger innanför själva diven */
              /* I vårt fall så ligger både cardFront och cardBack innanför vår inskickade div */
              var front = element.children[0]; /* Här hämtar vi kortets framsida */
              var back = element.children[1];  /* Här hämtar vi kortets baksida */

              if (front.classList.contains("flipped")){ /* Här kollar vi om kortets framsida har en klass som heter "flipped" */
                front.classList.replace("flipped", "flippBack") /* Här byter vi ut klassen "flipped" till klassen "flippBack" */
                back.classList.replace("flippBack", "flipped")  /* På baksidan gör vi tvärtom, byter ut "flippedBack" mot "flipped" */
              }
              else{
              
              }
            }

I funktionen ovan finns det bara kod för att vända så att baksidan syns, för att vända tillbaka krävs lite kod till
Som ni ser finns det en tom else-sats, där måste ni skriva lite egen kod för att vända tillbaka korten
Koden som behövs är väldigt lik den som finns i if-satsen, så försök klura ut vad som behöver ändras från den!

Nu borde koden fungera, men som bekant är det väldigt lätt att råka sriva fel på vägen så be om hjälp om det inte fungerar!
